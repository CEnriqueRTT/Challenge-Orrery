Create an Orrery Web App that Displays Near-Earth Objects

Hello world! We are a multidisciplinary team.

information about:

Planet Tech

Our team is working on an educational and interactive platform that aims to transform how the general public accesses and understands information about celestial bodies near Earth. Inspired by the idea of ​​creating an immersive experience, we have designed an interactive planetarium based on real NASA data, allowing users to explore the solar system and nearby objects, such as planets, asteroids, and comets. This platform will not only provide detailed information about these celestial bodies but will also include an artificial intelligence bot called Anab612, designed to answer user questions about space-related topics. In this way, we seek to combine the power of interactive visualization with instant access to scientific answers, creating a unique educational space.

Resources Used

Web Technologies

  • HTML: Defines the structure of the web application.
  • CSS: Styles and adapts the design for an engaging visual experience.
  • JavaScript: Adds interactivity and dynamic functionalities to the application.

3D Graphics

  • 3D Rendering: Allows the visualization of three-dimensional objects such as planets.
  • Animations: Simulates the orbital motion of celestial bodies.
  • Web Compatibility: Works efficiently in modern browsers.

Smart Chatbot

  • Technical Assistance: Offers help in code generation and troubleshooting.
  • Machine Learning: Provides answers based on context and prior knowledge.
  • Interactivity: Improves the user experience with personalized and real-time responses.

Nasa Data

  • Accurate Information: Provides up-to-date data on celestial bodies near Earth.
  • Open Access: Freely accessible resources to create accurate simulations.
  • Orrery Applications: Offers examples and tools for the development of the interactive model of the solar system.
